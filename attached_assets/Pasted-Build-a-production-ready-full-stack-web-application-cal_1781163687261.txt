Build a production-ready full-stack web application called **SkillGap AI – Intelligent Career Readiness Platform**.

GOAL:
Create a professional SaaS-style platform that analyzes resumes against job descriptions, identifies skill gaps using NLP and machine learning, provides ATS scoring, generates personalized learning roadmaps, validates recommendations using job market data, and presents everything through an interactive dashboard with complete source transparency.

IMPORTANT REQUIREMENTS:

1. The application must run without modification on:

   * Replit
   * Local Windows machine
   * Local Linux machine
   * Local Mac machine

2. The application must be GitHub-ready with:

   * Clean folder structure
   * README.md
   * requirements.txt
   * Environment variable support
   * Proper comments and documentation

3. The application must use:

   * Python
   * Flask
   * SQLite
   * Pandas
   * NumPy
   * Scikit-Learn
   * spaCy
   * Plotly
   * Bootstrap 5
   * ReportLab

4. Use modular architecture.
